msBatchRun.m outlines the order one would generally run the processing and analysis functions.

The SignleFunctions folder contains all analysis and processing function that run on a single data set stored in the MATLAB workspace.

The BatchFunctions folder contains functions that will run analysis and processing on all data sets located in the folder and subfolders of the data path sent in the functions argument.
